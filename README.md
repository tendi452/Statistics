'''
MEASURES OF CENTRALITY
centrality in data  typically describes where most of the data is.
Three most common centrality  mode, median, mean.
'''

'''
MEAN 
easiest measure of centrality.
also called the average.
it is the sum of all values diived by the number of values. 
'''

'''
MEDIAN
slightly less-common known measure of centrality
it is basically the middle term/value in the set of values.
it provides a good measure of what is typical in a set of value.
Unlike the mean, the median is independent of the extreme outlier in the data and the differences in the distribution of the data.
'''

'''
MODE
it is the most frequent value in the set of data. 
provides a good measure of the "typical" value in the set of values. 
the mode is better suited for working with categorical data.
'''
'''
MEASURES OF SPREAD
describes the "closeness" of the data.

VARIANCE
it is the absolute measure of how "spread out" the data is. 

STANDARD DEVIATION 
square root of the variance.
it is dependent on the variance.
it is commonly used when defining outliers.
'''
